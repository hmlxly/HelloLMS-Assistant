# 🎓 Hello LMS 助手 (智能网课助手)

![Version](https://img.shields.io/badge/Version-v6.14-blue.svg)
![Platform](https://img.shields.io/badge/Platform-Hello_LMS-orange.svg)

专为采用 **Hello LMS** 系统的韩国高校（如 DCU 大邱天主教大学）制作的网课自动播放助手。

## ✨ 主要功能

- 🚀 **全自动挂机**：自动播放、自动静音、自动点击下一集。
- 📺 **全面兼容**：支持原生视频和 YouTube 外部嵌入视频。
- 🛡️ **智能防打扰**：自动粉碎系统弹窗，自动修复面板跑出屏幕的问题。
- 🌐 **多语言界面**：内置中文、英文、韩文，一键无缝切换。
- 🔄 **云端更新**：一键检查更新，无需反复重新下载。

---

## 🛠️ 如何安装

1. 首先，在你的浏览器安装 **[Tampermonkey (油猴插件)](https://www.tampermonkey.net/)**。
2. 安装好插件后，👉 **[点击这里一键安装本助手](https://raw.githubusercontent.com/hmlxly/HelloLMS-Assistant/main/hello_lms_bot.user.js)**。

---

## 🚀 怎么使用

1. 登录你的学校网课系统，进入想要听的**课程页面**。
2. 页面会自动弹出助手面板（需要输入授权码解锁）。
3. 点击面板底部的 **[ 📡 扫描 ]** 按钮，让助手获取所有视频的进度。
4. 点击 **[ 🚀 启动 ]**，放开双手，它会自动帮你按顺序看完剩下的视频！

---

## 💡 常见问题

- **面板不见了？**
  看看网页右下角是不是有一个黄色的 `⚙️` 齿轮图标，点一下面板就会重新出来。
- **视频进度错乱或跳课？**
  点击面板底部的 `[ 🗑️ 重置 ]`，然后重新点击 `[ 📡 扫描 ]` 刷新进度即可。
- **不会自动放 YouTube 视频？**
  浏览器限制了外部视频自动播放。脚本会自动将视频**静音**来强制播放，请不要把浏览器最小化。
